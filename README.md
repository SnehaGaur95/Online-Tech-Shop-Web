# Online Tech Shop Web

## Overview

**Online Tech Shop Web** is an e-commerce platform designed to provide users with a smooth and secure online shopping experience for technology products. The system supports role-based access for **Customers, Managers, and Admins**, ensuring efficient management of products, users, and orders.

The platform is developed using **HTML, CSS, JavaScript, PHP, and JSON/MySQL**, enabling secure authentication, product browsing, order processing, and sales management.

---

## Features

### Customer (User) Features

* **Product Catalog**: Browse a wide range of tech products with search and filter options.
* **Shopping Cart**: Add or remove products, update quantities, and view cart summary.
* **Order Checkout**: Complete purchases through a secure checkout process.

---

### Manager Features

* **Add Product**: Add new tech products to the catalog.
* **Product List Management**: View, edit, or remove existing products.
* **Profile Management**: View and update manager profile details.

---

### Admin Features

* **Employee Management**: Add and manage employee records.
* **Product Management**: Add, edit, or remove products.
* **Customer Management**: View customer profiles and activity history.
* **Order Management**: Process and manage customer orders.
* **Sales Information**: Monitor sales data and generate reports.
* **Profile Management**: View and update admin profile details.

---

## Authentication System

* Secure **Login and Registration** using PHP session-based authentication.
* **Role-Based Access Control** for Admin, Manager, and Customer users.

## Installation Guide

### Prerequisites

* PHP Server (XAMPP, WAMP, or similar)
* MySQL Database
* Modern Web Browser

---

### Steps to Install

1. **Clone the Repository**

   ```bash
   git clone https://github.com/shishir786/Online-Tech-Shop.git
   ```

2. **Navigate to Project Directory**

   ```bash
   cd online_tech_shop-web
   ```

3. **Set Up Database**

   * Create a database in MySQL
   * Import the `online_tech_shop.sql` file
   * Required tables (users, products, orders, employees) will be created automatically

4. **Run the PHP Server**

   * Move the project folder to:

     * `htdocs` (XAMPP) or
     * `www` (WAMP)
   * Open browser and visit:

     ```
     http://localhost/online_tech_shop-web
     ```
